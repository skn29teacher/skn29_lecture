# RMSProp 옵티마이저 분석


### 전문 용어 사전 설명
본격적인 분석에 앞서, RMSProp의 핵심 개념을 정의합니다.

*   **지수 이동 평균 (Exponential Moving Average, EMA):** 오래된 과거의 데이터일수록 영향력을 지수적으로 감소시키고, 최근 데이터에 더 높은 가중치를 주어 평균을 내는 방식입니다.
*   **감쇠율 (Decay Rate, $\rho$):** 과거의 누적 정보를 얼마나 유지할지 결정하는 비율입니다. 보통 0.9나 0.99를 사용하며 '로'라고 읽습니다.
*   **적응적 학습률 (Adaptive Learning Rate):** 매개변수의 변화량에 따라 보폭을 스스로 조절하는 방식입니다.

---

### 1. 정의 및 배경 설명
**RMSProp**은 과거의 모든 기울기를 균등하게 누적하는 대신, 최근의 기울기 위주로 지수 이동 평균을 계산하여 학습률이 급격히 작아지는 것을 방지하고 지속적인 학습이 가능하게 만든 알고리즘입니다.

---

### 2. 전체 수식 제공 (표준형)
RMSProp은 지수 이동 평균을 이용해 기울기 제곱합을 갱신하는 단계와 매개변수를 업데이트하는 단계로 나뉩니다.

$$G_t = \rho G_{t-1} + (1 - \rho) (\nabla_{\theta} L(\theta_t, x^{(i)}, y^{(i)}))^2$$
$$\theta_{t+1} = \theta_t - \frac{\eta}{\sqrt{G_t + \epsilon}} \cdot \nabla_{\theta} L(\theta_t, x^{(i)}, y^{(i)})$$

**[변수 사전 정의]**
| 변수 | 의미 | 설명 |
| :--- | :--- | :--- |
| $G_t$ | 기울기 제곱의 이동 평균 | 최근의 기울기 변화량을 반영하는 변수 |
| $\rho$ | 감쇠율 | 과거 정보를 얼마나 희석할지 결정 (예: 0.9) |
| $\eta$ | 학습률 | 전체적인 업데이트 보폭 |
| $\nabla_{\theta} L$ | 현재 기울기 | 이번 단계의 기울기 값 |
| $\epsilon$ | 안전 상수 | 분모 0 방지 ($10^{-8}$) |

---

### 3. 수식 유도 과정 (데이터 대입 증명)

초기값 $\theta=2.0, G_0=0$, 학습률 $\eta=0.1$, 감쇠율 $\rho=0.9$ 상황에서 **기울기가 $10.0$으로 계속 큰 경우**를 가정하여 AdaGrad와 어떻게 다른지 증명합니다.

1.  **Step 1 (첫 업데이트)**:
    *   이동 평균 계산: $G_1 = (0.9 \cdot 0) + (1-0.9) \cdot 10^2 = 0 + 0.1 \cdot 100 = \mathbf{10.0}$
    *   조정된 학습률 적용: $\frac{0.1}{\sqrt{10.0}} \approx \mathbf{0.0316}$
    *   핸들 조절: $\theta_1 = 2.0 - (0.0316 \cdot 10.0) = \mathbf{1.684}$
    *   *증명*: AdaGrad($G=100$)보다 $G$값이 작게 유지되어 학습률이 덜 급격하게 줄어듭니다.

2.  **Step 2 (두 번째 업데이트)**:
    *   이동 평균 계산: $G_2 = (0.9 \cdot 10.0) + (0.1 \cdot 100) = 9 + 10 = \mathbf{19.0}$
    *   조정된 학습률 적용: $\frac{0.1}{\sqrt{19.0}} \approx \mathbf{0.0229}$
    *   핸들 조절: $\theta_2 = 1.684 - (0.0229 \cdot 10.0) = \mathbf{1.455}$
    *   *증명*: 과거의 $100$이 그대로 더해지는 것이 아니라 $0.9$배로 희석되었기 때문에, 학습률이 소멸하지 않고 적절히 유지됩니다.

---

### 4. 수치적 증명 (Dry Run)

지속적인 큰 기울기 상황에서 AdaGrad와 RMSProp의 누적값($G_t$) 차이를 비교합니다.

| 단계 | 현재 기울기 | AdaGrad 누적값 | RMSProp 누적값 | RMSProp 실질보폭 |
| :--- | :--- | :--- | :--- | :--- |
| **1** | 10.0 | 100.0 | **10.0** | 0.316 |
| **2** | 10.0 | 200.0 | **19.0** | 0.229 |
| **3** | 10.0 | 300.0 | **27.1** | 0.192 |

---

### 5. 코드 구현 및 데이터 증명
```python
# RMSProp의 지수 이동 평균 효과 증명 코드
import math

theta = 2.0
G = 0
rho = 0.9   # 감쇠율
eta = 0.1
grad = 10.0 # 큰 기울기
eps = 1e-8

for step in range(1, 4):
    G = rho * G + (1 - rho) * (grad**2)      # 1. 지수 이동 평균 갱신
    adaptive_eta = eta / math.sqrt(G + eps)   # 2. 적응적 학습률 계산
    theta = theta - adaptive_eta * grad      # 3. 업데이트
    print(f"Step {step}: 누적값G={G:.1f}, 실질보폭={adaptive_eta*grad:.3f}")
```

**[최종 증명 결론]**
데이터 대입 결과, RMSProp은 과거의 기울기 제곱을 **지수적으로 희석($\rho=0.1$ 만큼 반영)**시키기 때문에 누적값 $G$가 무한히 커지는 것을 막습니다. 이는 학습률이 너무 빨리 작아져서 학습이 중단되는 AdaGrad의 치명적인 단점을 해결했음을 수학적으로 증명합니다.
