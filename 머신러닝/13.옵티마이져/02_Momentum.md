# Momentum 옵티마이저 분석


### 전문 용어 사전 설명
본격적인 분석에 앞서, 모멘텀 알고리즘의 핵심 개념을 정의합니다.

*   **속도 (Velocity, $v$):** 매개변수가 현재 이동하고 있는 방향과 그 크기를 담고 있는 상태입니다. 물리적인 속도 개념과 같습니다.
*   **관성 계수 (Momentum Coefficient, $\gamma$):** 과거에 이동했던 속도를 얼마나 유지할지 결정하는 비율입니다. '감마'라고 읽으며 보통 0.9를 사용합니다.
*   **가속 (Acceleration):** 같은 방향으로 계속 이동할 때 속도가 점점 빨라지는 현상입니다.
*   **관성 (Inertia):** 기울기가 바뀌더라도 이전의 이동 방향을 유지하려는 성질입니다.

---

### 1. 정의 및 배경 설명
**Momentum** 옵티마이저는 단순 경사 하강법에 '관성'의 개념을 도입하여, 기울기가 일정한 방향으로 유지될 경우 가속도를 붙이고 방향이 급격히 바뀔 때는 진동을 억제하여 수렴 속도를 높이는 알고리즘입니다.

---

### 2. 전체 수식 제공 (표준형)
모멘텀의 업데이트 과정은 **속도 계산**과 **매개변수 업데이트**의 2단계로 이루어집니다.

$$v_t = \gamma v_{t-1} + \eta \nabla_{\theta} L(\theta_t, x^{(i)}, y^{(i)})$$
$$\theta_{t+1} = \theta_t - v_t$$

**[변수 사전 정의]**
| 변수 | 의미 | 설명 |
| :--- | :--- | :--- |
| $v_t$ | 현재 속도 | 이번 단계에서 이동할 방향과 크기 |
| $v_{t-1}$ | 이전 속도 | 직전 단계에서 가졌던 속도 |
| $\gamma$ | 관성 계수 | 이전 속도를 얼마나 보존할지 결정 (예: 0.9) |
| $\eta$ | 학습률 | 이번 단계의 기울기를 얼마나 반영할지 결정 |
| $\nabla_{\theta} L$ | 현재 기울기 | 현재 위치에서 가장 가파른 방향 |
| $\theta_t, \theta_{t+1}$ | 매개변수 | 현재 핸들 위치와 조절된 후의 위치 |

---

### 3. 수식 유도 과정 (데이터 대입 증명)

초기값 $\theta=10.0, v_0=0$, 보폭 $\eta=0.1$, 관성 $\gamma=0.9$ 상황에서 **기울기가 항상 $1.0$인 내리막길**을 내려간다고 가정합니다.

1.  **Step 1 (첫 번째 이동)**:
    *   속도 계산: $v_1 = (0.9 \cdot 0) + (0.1 \cdot 1.0) = \mathbf{0.1}$
    *   핸들 조절: $\theta_1 = 10.0 - 0.1 = \mathbf{9.9}$
    *   *증명*: 첫 단계는 일반 SGD와 동일하게 0.1만큼 이동합니다.

2.  **Step 2 (두 번째 이동 - 가속 시작)**:
    *   속도 계산: $v_2 = (0.9 \cdot 0.1) + (0.1 \cdot 1.0) = 0.09 + 0.1 = \mathbf{0.19}$
    *   핸들 조절: $\theta_2 = 9.9 - 0.19 = \mathbf{9.71}$
    *   *증명*: 이전 속도($0.1$)의 90%가 보존되어, 이동 거리가 **0.1에서 0.19로 증가**했습니다.

3.  **Step 3 (세 번째 이동 - 가속 누적)**:
    *   속도 계산: $v_3 = (0.9 \cdot 0.19) + (0.1 \cdot 1.0) = 0.171 + 0.1 = \mathbf{0.271}$
    *   핸들 조절: $\theta_3 = 9.71 - 0.271 = \mathbf{9.439}$
    *   *증명*: 이동 거리가 다시 **0.271**로 늘어났습니다. 관성에 의해 가속도가 붙음을 수치로 보여줍니다.

---

### 4. 수치적 증명 (Dry Run)

| 단계 | 이전 속도($v_{t-1}$) | 현재 기울기 | 계산된 새 속도($v_t$) | 결과 위치($\theta$) |
| :--- | :--- | :--- | :--- | :--- |
| **0** | - | - | 0 | 10.0 |
| **1** | 0 | 1.0 | **0.1** | 9.9 |
| **2** | 0.1 | 1.0 | **0.19** | 9.71 |
| **3** | 0.19 | 1.0 | **0.271** | 9.439 |

---

### 5. 코드 구현 및 데이터 증명
```python
# 모멘텀 가속 효과 증명 코드
theta = 10.0    # 초기 위치
v = 0           # 초기 속도
gamma = 0.9     # 관성 계수
eta = 0.1       # 학습률
grad = 1.0      # 일정한 기울기 가정

for step in range(1, 4):
    v = gamma * v + eta * grad  # 1. 속도 업데이트
    theta = theta - v           # 2. 위치 업데이트
    print(f"Step {step}: 이동거리={v:.3f}, 현재위치={theta:.3f}")
```

**[최종 증명 결론]**
데이터 대입 결과, 모멘텀 알고리즘은 **이동 거리가 0.1 → 0.19 → 0.271로 점진적으로 증가**함을 보여줍니다. 이는 똑같은 기울기 환경에서 매번 0.1씩만 이동하는 일반 SGD보다 훨씬 빠르게 최적점에 도달할 수 있음을 수학적, 수치적으로 증명합니다.
