# AdaGrad 옵티마이저 분석


### 전문 용어 사전 설명
본격적인 분석에 앞서, AdaGrad의 핵심 개념을 정의합니다.

*   **누적 기울기 제곱 (Accumulated Gradient Squares, $G$):** 과거부터 현재까지 발생한 모든 기울기 값들을 각각 제곱하여 계속 더해나간 값입니다.
*   **적응적 학습률 (Adaptive Learning Rate):** 모든 매개변수에 동일한 보폭을 적용하지 않고, 많이 변한 변수는 작게, 적게 변한 변수는 크게 보폭을 자동으로 조절하는 방식입니다.
*   **엡실론 (Epsilon, $\epsilon$):** 수식에서 분모가 0이 되는 것을 방지하기 위해 더해주는 아주 작은 상수(보통 $10^{-8}$)입니다.

---

### 1. 정의 및 배경 설명
**AdaGrad**는 각각의 매개변수마다 맞춤형 학습률을 적용하는 알고리즘으로, 그동안 크게 업데이트된 매개변수는 학습률을 낮추고 적게 업데이트된 매개변수는 학습률을 유지하여 정교한 수렴을 돕는 옵티마이저입니다.

---

### 2. 전체 수식 제공 (표준형)
AdaGrad는 기울기의 제곱합을 관리하는 단계와 이를 이용해 매개변수를 업데이트하는 단계로 나뉩니다.

$$G_t = G_{t-1} + (\nabla_{\theta} L(\theta_t, x^{(i)}, y^{(i)}))^2$$
$$\theta_{t+1} = \theta_t - \frac{\eta}{\sqrt{G_t + \epsilon}} \cdot \nabla_{\theta} L(\theta_t, x^{(i)}, y^{(i)})$$

**[변수 사전 정의]**
| 변수 | 의미 | 설명 |
| :--- | :--- | :--- |
| $G_t$ | 누적 기울기 제곱 | 현재까지의 기울기 변화량을 저장하는 변수 |
| $\nabla_{\theta} L$ | 현재 기울기 | 이번 단계에서 계산된 기울기 값 |
| $\eta$ | 기본 학습률 | 전체적인 보폭을 결정하는 초기값 |
| $\theta_t, \theta_{t+1}$ | 매개변수 | 현재 상태와 업데이트된 후의 상태 |
| $\epsilon$ | 안전 상수 | 분모가 0이 됨을 방지 (예: $10^{-8}$) |

---

### 3. 수식 유도 과정 (데이터 대입 증명)

초기값 $\theta=2.0, G_0=0$, 학습률 $\eta=0.1$ 상황에서 **기울기가 $10.0$으로 매우 큰 환경**을 가정하여 학습률이 어떻게 줄어드는지 증명합니다.

1.  **Step 1 (첫 업데이트)**:
    *   기울기 제곱 누적: $G_1 = 0 + (10.0)^2 = \mathbf{100.0}$
    *   조정된 학습률 계산: $\frac{0.1}{\sqrt{100.0}} = \frac{0.1}{10} = \mathbf{0.01}$
    *   핸들 조절: $\theta_1 = 2.0 - (0.01 \cdot 10.0) = \mathbf{1.9}$
    *   *증명*: 초기 학습률 $0.1$이 큰 기울기 때문에 즉시 **0.01로 10배 축소**되었습니다.

2.  **Step 2 (두 번째 업데이트)**:
    *   기울기 제곱 누적: $G_2 = 100.0 + (10.0)^2 = \mathbf{200.0}$
    *   조정된 학습률 계산: $\frac{0.1}{\sqrt{200.0}} \approx \frac{0.1}{14.14} \approx \mathbf{0.007}$
    *   핸들 조절: $\theta_2 = 1.9 - (0.007 \cdot 10.0) = \mathbf{1.83}$
    *   *증명*: 학습이 진행될수록 학습률이 **0.007로 더 작아지며** 매우 조심스럽게 수렴합니다.

---

### 4. 수치적 증명 (Dry Run)

| 단계 | 현재 기울기 | 누적값($G_t$) | 실질 학습률 ($\eta / \sqrt{G}$) | 이동 거리 |
| :--- | :--- | :--- | :--- | :--- |
| **1** | 10.0 | 100.0 | 0.010 | 0.100 |
| **2** | 10.0 | 200.0 | 0.007 | 0.070 |
| **3** | 10.0 | 300.0 | 0.005 | 0.057 |

---

### 5. 코드 구현 및 데이터 증명
```python
# AdaGrad의 학습률 감쇠 효과 증명 코드
import math

theta = 2.0
G = 0
eta = 0.1
grad = 10.0  # 큰 기울기 가정
eps = 1e-8

for step in range(1, 4):
    G = G + grad**2                  # 1. 기울기 제곱 누적
    adaptive_eta = eta / math.sqrt(G + eps) # 2. 적응적 학습률 계산
    theta = theta - adaptive_eta * grad     # 3. 업데이트
    print(f"Step {step}: 실질학습률={adaptive_eta:.4f}, 이동거리={adaptive_eta*grad:.4f}")
```

**[최종 증명 결론]**
데이터 대입 결과, AdaGrad는 **기울기가 클수록 학습률을 스스로 낮추어(0.010 → 0.007 → 0.005)** 모델이 발산하지 않고 안정적으로 수렴하게 만듭니다. 이는 수동으로 학습률을 조정할 필요 없이 알고리즘이 스스로 보폭을 결정함을 증명합니다.
