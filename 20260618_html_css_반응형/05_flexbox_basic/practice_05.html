2. 교차축 정렬 데모 (align-items)

세로 방향(교차축)의 높이가 다른 아이템들을 수직 정렬하는 설정을 비교합니다.

align-items: stretch (부모 높이로 강제 늘림)
1 (높이없음)
2 (높이없음)
3 (높이없음)
align-items: center (수직 중앙 정렬)
1 (50px)
2 (80px)
3 (60px)
align-items: flex-start(위) / flex-end(아래)
start
self-end(이탈)
start

3. 팽창 비율 제어 (flex-grow)

잉여 여백을 나누어 갖는 아이템들의 크기 비율을 관찰합니다.

지분율 1 (grow: 1)
지분율 2 (grow: 2)
지분율 1 (grow: 1)
