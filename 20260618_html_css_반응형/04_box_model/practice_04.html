1. 박스 모델 구성요소 데모 (Box Model Playground)

패딩, 보더, 마진이 실제로 레이아웃에 어떻게 부피를 형성하고 영역을 차지하는지 눈으로 확인합니다.

content-box 모델

너비 200px + 패딩 20px + 테두리 5px = 총 250px 부피 차지

border-box 모델

너비 200px 설정 고정 (내부에 패딩, 테두리가 포함됨) = 총 200px 부피 유지
