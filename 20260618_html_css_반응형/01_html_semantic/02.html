4. 1일차 주요 웹 표준 용어 정리

HTML (HyperText Markup Language)
웹 브라우저 상에 웹 페이지의 뼈대와 내용을 그리기 위한 구조적 마크업 언어입니다.
CSS (Cascading Style Sheets)
HTML 문서에 디자인, 스타일, 레이아웃, 크기, 애니메이션 등을 입히는 스타일 기술 언어입니다.
viewport (뷰포트)
모바일 등 디지털 디바이스 장치에서 웹 화면이 가독성 있게 표현되도록 크기와 비율을 제어하는 메타 장치입니다.
