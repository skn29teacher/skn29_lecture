1-D. minmax() — 최소·최대 크기 지정

섹션 4: grid-column / grid-row — span으로 셀 병합

grid-column: span N과 grid-row: span N으로 여러 셀을 병합하는 매거진 스타일 레이아웃을 구성합니다.

A
grid-column: 1 / span 2
가로 2칸 차지
B
grid-row: 1 / span 2
세로 2칸 차지
C
grid-row: 1 / span 3
세로 3칸 차지
D
기본 크기
1×1 셀
E
grid-row: span 2
세로 2칸 차지
F
기본 크기
1×1 셀

📋 각 아이템 CSS 속성

아이템 grid-column grid-row 설명
A 1 / span 2 auto 가로 2열 차지
B auto 1 / span 2 세로 2행 차지
C auto 1 / span 3 세로 3행 차지
D auto auto 기본 1×1 셀
E auto span 2 세로 2행 차지
F auto auto 기본 1×1 셀
