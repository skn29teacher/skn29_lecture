# 세션 3: 데이터 로드 및 저장

## 1. 데이터 입출력 개요

### 1.1 Pandas가 지원하는 파일 형식
- **CSV**: 쉼표로 구분된 텍스트 파일
- **Excel**: .xlsx, .xls 파일
- **JSON**: JavaScript Object Notation
- **HTML**: 웹 페이지의 표
- **SQL**: 데이터베이스
- **Pickle**: Python 객체 직렬화
- **Parquet**: 컬럼 기반 저장 형식

### 1.2 주요 입출력 메서드
```
읽기                     쓰기
-----------------       -----------------
pd.read_csv()          df.to_csv()
pd.read_excel()        df.to_excel()
pd.read_json()         df.to_json()
pd.read_html()         df.to_html()
pd.read_sql()          df.to_sql()
```

## 2. CSV 파일 다루기

### 2.1 CSV 파일 읽기

#### 기본 읽기
```python
df = pd.read_csv('data.csv')
```

#### 주요 파라미터
```python
df = pd.read_csv(
    'data.csv',
    sep=',',                    # 구분자 (기본값: 쉼표)
    header=0,                   # 헤더 행 번호 (기본값: 0)
    index_col=0,                # 인덱스로 사용할 열
    names=['A', 'B', 'C'],      # 컬럼명 지정
    usecols=['A', 'B'],         # 특정 컬럼만 읽기
    dtype={'A': int, 'B': str}, # 데이터 타입 지정
    encoding='utf-8',           # 인코딩 (한글: 'cp949', 'utf-8')
    na_values=['NA', 'null'],   # 결측치로 인식할 값
    skiprows=2,                 # 처음 n개 행 건너뛰기
    nrows=100                   # n개 행만 읽기
)
```

#### 인코딩 문제 해결
```python
# Windows에서 한글 깨짐 시
df = pd.read_csv('data.csv', encoding='cp949')

# UTF-8 BOM 문제
df = pd.read_csv('data.csv', encoding='utf-8-sig')
```

### 2.2 CSV 파일 쓰기

#### 기본 저장
```python
df.to_csv('output.csv')
```

#### 주요 파라미터
```python
df.to_csv(
    'output.csv',
    sep=',',                # 구분자
    index=False,            # 인덱스 저장 여부
    header=True,            # 헤더 저장 여부
    encoding='utf-8',       # 인코딩
    columns=['A', 'B'],     # 특정 컬럼만 저장
    na_rep='NA',            # 결측치 표현
    float_format='%.2f'     # 소수점 포맷
)
```

## 3. Excel 파일 다루기

### 3.1 Excel 파일 읽기

#### 단일 시트 읽기
```python
df = pd.read_excel('data.xlsx')
```

#### 특정 시트 읽기
```python
# 시트 이름으로
df = pd.read_excel('data.xlsx', sheet_name='Sheet1')

# 시트 인덱스로
df = pd.read_excel('data.xlsx', sheet_name=0)

# 여러 시트 읽기
dfs = pd.read_excel('data.xlsx', sheet_name=['Sheet1', 'Sheet2'])
# 또는 모든 시트
dfs = pd.read_excel('data.xlsx', sheet_name=None)
```

#### 주요 파라미터
```python
df = pd.read_excel(
    'data.xlsx',
    sheet_name='Sheet1',        # 시트 이름
    header=0,                    # 헤더 행
    index_col=0,                 # 인덱스 열
    usecols='A:C',              # 열 범위 지정
    skiprows=2,                  # 건너뛸 행
    nrows=100,                   # 읽을 행 수
    na_values=['NA']             # 결측치 값
)
```

### 3.2 Excel 파일 쓰기

#### 단일 시트 저장
```python
df.to_excel('output.xlsx', index=False)
```

#### 여러 시트 저장
```python
with pd.ExcelWriter('output.xlsx') as writer:
    df1.to_excel(writer, sheet_name='Sheet1', index=False)
    df2.to_excel(writer, sheet_name='Sheet2', index=False)
    df3.to_excel(writer, sheet_name='Sheet3', index=False)
```

#### 주요 파라미터
```python
df.to_excel(
    'output.xlsx',
    sheet_name='Sheet1',    # 시트 이름
    index=False,            # 인덱스 저장 여부
    header=True,            # 헤더 저장 여부
    columns=['A', 'B'],     # 특정 컬럼만 저장
    na_rep='NA',            # 결측치 표현
    float_format='%.2f',    # 소수점 포맷
    engine='openpyxl'       # 엔진 (openpyxl, xlsxwriter)
)
```

## 4. JSON 파일 다루기

### 4.1 JSON 파일 읽기

#### 기본 읽기
```python
df = pd.read_json('data.json')
```

#### JSON 형식
```python
# orient 파라미터
# 'split': 인덱스, 컬럼, 데이터가 분리
# 'records': 레코드 배열
# 'index': 인덱스가 키
# 'columns': 컬럼이 키
# 'values': 값만 (2차원 배열)

df = pd.read_json('data.json', orient='records')
```

### 4.2 JSON 파일 쓰기

```python
# 기본
df.to_json('output.json')

# 형식 지정
df.to_json('output.json', orient='records')

# 들여쓰기 (가독성)
df.to_json('output.json', orient='records', indent=4)

# 한글 깨짐 방지
df.to_json('output.json', force_ascii=False)
```

## 6. 데이터베이스 연동

### 6.1 SQLite 데이터베이스

```python
import sqlite3

# 연결
conn = sqlite3.connect('database.db')

# 테이블 읽기
df = pd.read_sql('SELECT * FROM table_name', conn)

# 쿼리 실행
df = pd.read_sql('SELECT * FROM table_name WHERE age > 30', conn)

# 데이터 저장
df.to_sql('table_name', conn, if_exists='replace', index=False)
# if_exists: 'fail', 'replace', 'append'

# 연결 종료
conn.close()
```

### 6.2 기타 데이터베이스 (MySQL, PostgreSQL 등)

```python
from sqlalchemy import create_engine

# MySQL 연결
engine = create_engine('mysql://user:password@localhost/database')

# PostgreSQL 연결
engine = create_engine('postgresql://user:password@localhost/database')

# 읽기
df = pd.read_sql('SELECT * FROM table_name', engine)

# 쓰기
df.to_sql('table_name', engine, if_exists='replace', index=False)
```

## 7. 기타 파일 형식

### 7.1 Pickle (Python 객체)

```python
# 저장 (가장 빠름)
df.to_pickle('data.pkl')

# 읽기

```

### 7.2 Parquet (빅데이터 용)

```python
# 저장
df.to_parquet('data.parquet')

# 읽기
df = pd.read_parquet('data.parquet')
```

### 7.3 텍스트 파일 (Tab-separated)

```python
# 탭 구분
df = pd.read_csv('data.txt', sep='\t')

# 저장
df.to_csv('output.txt', sep='\t', index=False)
```

## 8. 대용량 데이터 처리

## 9. 실전 팁

### 9.1 파일 존재 확인

```python
import os

if os.path.exists('data.csv'):
    df = pd.read_csv('data.csv')
else:
    print("파일이 없습니다.")
```

### 9.2 여러 파일 병합

```python
import glob

# 같은 구조의 여러 CSV 파일 합치기
files = glob.glob('data/*.csv')
dfs = [pd.read_csv(f) for f in files]
df = pd.concat(dfs, ignore_index=True)
```

### 9.3 압축 파일 직접 읽기
