## 1. 인덱스(Index) 이해

### 1.1 인덱스란?
- 각 행을 식별하는 레이블
- 데이터 접근과 정렬의 기준
- 고유할 필요는 없지만 고유한 것이 일반적
- 불변(immutable) 객체

### 1.2 인덱스의 종류
- **RangeIndex**: 0부터 시작하는 정수 (기본값)
- **Int64Index**: 정수 인덱스
- **DatetimeIndex**: 날짜/시간 인덱스
- **MultiIndex**: 계층적 인덱스
- **CategoricalIndex**: 카테고리 인덱스

## 2. 인덱스 기본 조작

### 2.1 인덱스 확인
```python
df.index          # 인덱스 객체
df.index.name     # 인덱스 이름
df.index.values   # 인덱스 값
```

### 2.2 인덱스 설정
```python
# 특정 컬럼을 인덱스로
df.set_index('column_name')

# 여러 컬럼을 인덱스로 (MultiIndex)
df.set_index(['col1', 'col2'])

# inplace로 원본 수정
df.set_index('column_name', inplace=True)
```

### 2.3 인덱스 재설정
```python
# 인덱스를 컬럼으로 되돌리기
df.reset_index()

# 인덱스 삭제
df.reset_index(drop=True)
```

### 2.4 인덱스 이름 변경
```python
df.index.name = 'new_name'

# rename 메서드
df.rename_axis('new_name')
```

## 3. 인덱스를 이용한 데이터 접근

### 3.1 loc - 레이블 기반 인덱싱
```python
# 단일 행
df.loc[label]

# 여러 행
df.loc[[label1, label2, label3]]

# 슬라이싱 (끝 포함!)
df.loc[label1:label3]

# 행과 열 동시 선택
df.loc[label, 'column']
df.loc[label1:label3, ['col1', 'col2']]

# 모든 행, 특정 열
df.loc[:, 'column']
```

### 3.2 iloc - 위치 기반 인덱싱
```python
# 단일 행
df.iloc[0]

# 여러 행
df.iloc[[0, 2, 4]]

# 슬라이싱 (끝 미포함!)
df.iloc[0:5]

# 행과 열 동시 선택
df.iloc[0, 1]
df.iloc[0:3, 0:2]

# 모든 행, 특정 열
df.iloc[:, 1]
```

### 3.3 at과 iat - 단일 값 빠른 접근
```python
# 레이블 기반
df.at[label, 'column']

# 위치 기반
df.iat[0, 1]
```

## 4. 데이터 필터링 기본

### 4.1 Boolean 인덱싱
```python
# 단일 조건
df[df['age'] > 30]
df[df['name'] == 'Kim']

# 복합 조건 (AND: &)
df[(df['age'] > 25) & (df['salary'] > 4000)]

# 복합 조건 (OR: |)
df[(df['city'] == 'Seoul') | (df['city'] == 'Busan')]

# NOT 조건 (~)
df[~(df['age'] < 30)]
```

### 4.2 isin() 메서드
```python
# 여러 값 중 하나라도 포함
df[df['city'].isin(['Seoul', 'Busan', 'Incheon'])]

# 제외
df[~df['city'].isin(['Seoul', 'Busan'])]
```

### 4.3 between() 메서드
```python
# 범위 내 값
df[df['age'].between(25, 35)]

# inclusive 파라미터
df[df['age'].between(25, 35, inclusive='both')]  # 양쪽 포함 (기본값)
df[df['age'].between(25, 35, inclusive='neither')]  # 양쪽 제외
```

## 5. 문자열 필터링

### 5.1 문자열 메서드
```python
# 포함 여부
df[df['name'].str.contains('Kim')]

# 시작 문자열
df[df['name'].str.startswith('K')]

# 끝 문자열
df[df['name'].str.endswith('son')]

# 대소문자 무시
df[df['name'].str.contains('kim', case=False)]

# 정규표현식
df[df['email'].str.contains(r'\\w+@\\w+\\.com')]
```

### 5.2 문자열 변환
```python
df['name'].str.upper()     # 대문자
df['name'].str.lower()     # 소문자
df['name'].str.strip()     # 공백 제거
df['name'].str.replace('old', 'new')  # 치환
```

## 6. 결측치 필터링

### 6.1 결측치 확인
```python
df.isnull()       # 결측치면 True
df.notnull()      # 결측치가 아니면 True
df.isnull().sum()  # 각 컬럼의 결측치 개수
```

### 6.2 결측치로 필터링
```python
# 결측치가 있는 행
df[df['column'].isnull()]

# 결측치가 없는 행
df[df['column'].notnull()]

# 특정 컬럼에 결측치가 하나라도 있는 행
df[df[['col1', 'col2']].isnull().any(axis=1)]

# 모든 값이 결측치인 행
df[df.isnull().all(axis=1)]
```

## 7. query() 메서드

### 7.1 기본 사용
```python
# 문자열 표현식으로 필터링
df.query('age > 30')
df.query('age > 30 and salary > 4000')
df.query('city == "Seoul" or city == "Busan"')
```

### 7.2 변수 사용
```python
min_age = 25
df.query('age > @min_age')

cities = ['Seoul', 'Busan']
df.query('city in @cities')
```

### 7.3 문자열 메서드
```python
df.query('name.str.contains("Kim")')
df.query('name.str.startswith("K")')
```

## 8. 조건부 값 변경

### 8.1 loc를 이용한 변경
```python
# 조건에 맞는 행의 특정 컬럼 값 변경
df.loc[df['age'] > 30, 'category'] = 'Senior'

# 여러 컬럼 동시 변경
df.loc[df['age'] > 30, ['category', 'status']] = ['Senior', 'Active']
```

### 8.2 numpy.where()
```python
import numpy as np

# 조건에 따라 값 할당
df['category'] = np.where(df['age'] > 30, 'Senior', 'Junior')

# 여러 조건
df['grade'] = np.where(df['score'] >= 90, 'A',
                       np.where(df['score'] >= 80, 'B',
                               np.where(df['score'] >= 70, 'C', 'D')))
```

### 8.3 pandas.cut() - 구간 나누기
```python
# 동일 너비 구간
df['age_group'] = pd.cut(df['age'], bins=3, labels=['Young', 'Middle', 'Old'])

# 사용자 정의 구간
bins = [0, 20, 30, 40, 100]
labels = ['10s', '20s', '30s', '40+']
df['age_group'] = pd.cut(df['age'], bins=bins, labels=labels)
```

### 8.4 pandas.qcut() - 분위수 나누기
```python
# 4분위로 나누기
df['quartile'] = pd.qcut(df['salary'], q=4, labels=['Q1', 'Q2', 'Q3', 'Q4'])
```

## 9. 샘플링

### 9.1 sample() 메서드
```python
# n개 무작위 샘플
df.sample(n=10)

# 비율로 샘플링
df.sample(frac=0.2)  # 20%

# 복원 추출
df.sample(n=10, replace=True)

# 시드 고정 (재현성)
df.sample(n=10, random_state=42)
```

### 9.2 nlargest(), nsmallest()
```python
# 상위 n개
df.nlargest(10, 'salary')

# 하위 n개
df.nsmallest(10, 'age')

# 여러 컬럼 기준
df.nlargest(10, ['salary', 'age'])
```

## 10. 중복 데이터 처리

### 10.1 중복 확인
```python
# 중복 행 확인
df.duplicated()

# 특정 컬럼 기준 중복
df.duplicated(subset=['name'])

# 중복 개수
df.duplicated().sum()
```

### 10.2 중복 제거
```python
# 중복 제거
df.drop_duplicates()

# 특정 컬럼 기준
df.drop_duplicates(subset=['name'])

# 첫 번째/마지막 유지
df.drop_duplicates(keep='first')  # 기본값
df.drop_duplicates(keep='last')
df.drop_duplicates(keep=False)  # 모두 제거
```

## 11. 날짜/시간 인덱스

### 11.1 DatetimeIndex 생성
```python
# 날짜 컬럼을 인덱스로
df['date'] = pd.to_datetime(df['date'])
df.set_index('date', inplace=True)

# 날짜 범위 생성
dates = pd.date_range('2024-01-01', periods=365, freq='D')
df.index = dates
```

### 11.2 날짜로 필터링
```python
# 특정 날짜
df.loc['2024-01-01']

# 날짜 범위
df.loc['2024-01-01':'2024-01-31']

# 월별
df.loc['2024-01']

# 연도별
df.loc['2024']
```

### 11.3 시간 관련 속성
```python
df.index.year
df.index.month
df.index.day
df.index.dayofweek  # 요일 (0=월요일)
df.index.quarter    # 분기
```

## 12. MultiIndex (계층적 인덱스)

### 12.1 MultiIndex 생성
```python
# 여러 컬럼을 인덱스로
df.set_index(['city', 'department'])

# 직접 생성
arrays = [['서울', '서울', '부산', '부산'],
          ['영업', '개발', '영업', '개발']]
index = pd.MultiIndex.from_arrays(arrays, names=['city', 'dept'])
```

### 12.2 MultiIndex 접근
```python
# 외부 인덱스
df.loc['서울']

# 특정 조합
df.loc[('서울', '영업')]

# 여러 조합
df.loc[[('서울', '영업'), ('부산', '개발')]]

# xs() 메서드
df.xs('서울', level='city')
df.xs('영업', level='dept')
```
