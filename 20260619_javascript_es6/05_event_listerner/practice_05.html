2. event.target vs event.currentTarget 시각 대조

버튼 내부에 배치된 span 영역을 직접 마우스로 조준하여 누를 때, 실제 하위 충돌 지점(`event.target`)과 이벤트가 걸린 실제 주체(`event.currentTarget`)의 차이를 추적합니다.

감시자 영역 (div#event-parent-container)
