DOM 조작 및 XSS 보안 실증

동적 DOM 생성 및 삭제 API를 습득하고, XSS 보안 위협 실증 및 DocumentFragment 최적화를 테스트합니다.

2. 실시간 알림 센터 노드 라이프사이클 제어

`createElement`를 통해 메모리 노드를 동적으로 빌드하고, `prepend`를 이용해 최신 메시지를 상단에, `append`를 이용해 맨 하단에 덧붙이고, `remove`로 개별 요소를 소멸시킵니다.

3. DocumentFragment 기반 리플로우(Reflow) 최적화 렌더링

다량의 엘리먼트를 하나씩 브라우저 돔에 삽입하여 리플로우 연산 부하를 매번 일으키는 구식 방식과, 가상 버퍼 공간에 취합해 단 1회만 마운트하는 최적화 성능 격차를 이해합니다.
