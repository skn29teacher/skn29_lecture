## 1. Windows Nginx 특징, 통신제약
- UNIX 도메인 소켓 지원 불가
- TCP 루프백 통신 대체 : window환경에서 Nginx와 WAS(Waitress) 중계를 하려면 반드시 로컬호스인 127.0.0.1의 특정 TCP포트(8000)를 사용해 상호 통신

## 2. 연동흐름
브라우져 - Nginx(window) : 80 에서 대기 -> Waitress : 8000 에서 대기(Django 실행)

## 3. nginx 폴더 및 환경설정 파일 작성
nginx 폴더는 프로젝트 폴더와 동일한 공간에 작성
