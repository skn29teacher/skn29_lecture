# 📄 산출물 2: 인공지능 학습 결과서

## 1. 모델링 전략

### 1-1. 평가 지표 선정
| 지표 | 선정 이유 |
|------|----------|
| Recall (재현율) | 이탈 고객을 놓치는 것이 가장 큰 손실 → 최우선 지표 |
| F1-Score | Precision과 Recall의 균형 |
| ROC-AUC | 임계값(threshold)에 독립적인 전반적 성능 |
| Accuracy | 참고용 (불균형 데이터에서 단독 사용 지양) |

> ⚠️ 이탈 예측에서 False Negative(이탈인데 유지로 예측)가 더 위험
> → Recall을 최우선으로, F1-Score를 보조 지표로 설정

### 1-2. 후보 모델 선정
| 모델 | 유형 | 선정 근거 |
|------|------|----------|
| Logistic Regression | 머신러닝 (선형) | 기준선(Baseline) 모델, 해석 용이 |
| Random Forest | 머신러닝 (앙상블) | 불균형 데이터에 강건, 특성 중요도 제공 |
| XGBoost | 머신러닝 (부스팅) | 분류 성능 우수, 이탈 예측 업계 표준 |
| MLP (PyTorch) | 딥러닝 | 비선형 패턴 학습, 복잡한 특성 관계 포착 |

### 1-3. 실험 계획
- 각 모델에 기본 하이퍼파라미터로 베이스라인 성능 측정
- 최고 성능 모델에 하이퍼파라미터 튜닝 적용
- 최종 모델 선정 기준: Test Recall ≥ 0.80 & F1-Score 최고
