### 입력 데이터 스펙

- 특성 순서: feature_list.json 참조

### 예측값 해석
| 출력값 | 의미 |
|--------|------|
| 0 | 이탈하지 않음 (유지) |
| 1 | 이탈 예측 |
| predict_proba[:,1] | 이탈 확률 (0~1) |

### 모델 로드 및 예측 예시
```python
import pickle, json
import pandas as pd

# 모델 및 스케일러 로드
with open('best_model.pkl', 'rb') as f:
    model = pickle.load(f)
with open('scaler.pkl', 'rb') as f:
    scaler = pickle.load(f)
with open('feature_list.json', 'r') as f:
    features = json.load(f)

# 신규 고객 데이터 예측
new_customer = pd.DataFrame([{...}])  # 특성값 입력
new_customer_scaled = scaler.transform(new_customer[features])

prediction = model.predict(new_customer_scaled)
probability = model.predict_proba(new_customer_scaled)[:, 1]

print(f"이탈 예측: {'이탈' if prediction[0]==1 else '유지'}")
print(f"이탈 확률: {probability[0]:.2%}")
```
